# Dynamic Hospital Performance Dashboard

## Project Overview
Dynamic Hospital Performance Dashboard is a simple system that helps hospitals monitor their performance in real time.  
It shows important hospital data and gives alerts when there is a problem.

## Features
- Monitor hospital KPIs
- Detect unusual changes in data
- Generate simple AI insights
- Suggest corrective actions
- Role-based dashboards for hospital staff

## KPIs Monitored
- Bed Occupancy Rate
- OPD Waiting Time
- Billing Collection Rate
- Lab Test Turnaround Time
- Patient Satisfaction Score

## Technology Used
- Python
- Pandas
- Streamlit
- Plotly
- AI/LLM for insights

## How It Works
1. The system collects hospital data.
2. It compares today's data with previous data.
3. If a KPI changes too much, an alert is generated.
4. The system explains the problem and suggests actions.

## Benefits
- Real-time hospital monitoring
- Faster decision making
- Better patient service
- Improved hospital efficiency
